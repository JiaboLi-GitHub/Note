# Vulkan 学习笔记

## 1. 着色器通讯

### 1.1 顶点属性

基于顶点数组对象（VAO）和顶点缓冲对象（VBO）向顶点着色器传递数据。仅顶点着色器可读。

### 1.2 描述符

描述符用于在着色器之间传递数据，主要包括：

- **纹理（Texture）**：用于传递图像数据
- **统一缓冲（UBO）**：Uniform Buffer Object，用于传递常量数据
- **存储缓冲（SSBO）**：Storage Buffer Object，用于可读写的缓冲数据

### 1.3 推送常量

Push Constants 是极轻量的通讯方式，无需缓冲或描述符，适合传递少量频繁变化的数据。

### 1.4 管线常量

管线常量是编译时常量，CPU无法动态修改，仅在创建管线时可以赋值。

### 1.5 共享内存

着色器之间的共享内存机制（通常用于计算着色器）。

### 1.6 帧缓冲

帧缓冲用于存储渲染结果，可以作为后续渲染的输入。
